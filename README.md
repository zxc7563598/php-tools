# Tools

Hejunjie Tools 是一个 PHP 工具扩展库，封装了我在工作中常用的各种操作方法。该库包含以下功能模块：

- **字符串操作类**：提供常用的字符串处理方法。
- **数组操作类**：封装了高效的数组处理函数。
- **文件操作类**：简化文件读写和处理。
- **网络请求操作类**：方便的 HTTP 请求封装。
- **图片操作类**：处理图片相关的操作。
- **手机号运营商查询**：根据手机号获取对应的运营商信息。
- **身份证号户籍地查询**：根据身份证号获取户籍地信息。

该库后续将持续更新，添加更多实用功能。欢迎大家提供建议和反馈，我会根据大家的意见实现新的功能，共同提升开发效率。
